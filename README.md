# 📊 Sales Performance Dashboard (2020-2022)

## 🌟 Project Overview
This project presents a comprehensive **Sales Performance Dashboard** built using Power BI. It analyzes sales data from a technology company (Techno Sales) over a three-year period (2020-2022), providing actionable insights into revenue trends, regional performance, and product profitability.

---

## 📈 Key Metrics & KPIs
- **Total Sales:** $3.22 Million
- **Total Profit:** $286.40 Thousand
- **Total Orders:** 5,009
- **Analyzed Period:** 2020 - 2022

---

## 🖥️ Dashboard Features

### 1. Executive Summary
- Interactive KPI cards for at-a-glance performance tracking.
- Monthly sales and profit trends to identify seasonal patterns.
- Comparison of current vs. previous year performance.

### 2. Regional Analysis
- Geographic distribution of sales.
- Top-performing regions and states.
- Sales breakdown by customer segments (Consumer, Corporate, Home Office).

### 3. Product & Category Insights
- Treemap visualization for Sales by Product Category and Sub-Category.
- Identifying high-margin products and underperforming categories.
- Slicers for deep-diving into specific years, quarters, and regions.

---

## 🛠️ Tech Stack & Tools
- **Power BI Desktop:** Dashboard design and visualization.
- **Power Query:** Data cleaning, transformation, and ETL processes.
- **DAX (Data Analysis Expressions):** Calculated measures and complex business logic.
- **Excel:** Primary data source.

---

## 📂 Repository Structure
- `Data/`: Contains the raw Excel dataset.
- `Reports/`: Contains the `.pbix` Power BI report file.
- `Screenshots/`: High-resolution dashboard visuals.

## 🚀 How to View
1. Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
2. Clone this repository: `git clone https://github.com/YOUR_USERNAME/Power-BI-Sales-Dashboard-2020-22.git`
3. Open the `.pbix` file from the `Reports/` folder.
4. Ensure the data source path is updated to point to the file in the `Data/` folder.
